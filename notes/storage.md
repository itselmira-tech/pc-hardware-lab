# Storage (HDD / SSD)

## What is Storage?

Storage is the component that keeps data permanently on a computer.

## HDD (Hard Disk Drive)

- Uses magnetic disks
- Lower speed
- More mechanical parts
- Usually cheaper

## SSD (Solid State Drive)

- Uses flash memory
- Faster than HDD
- No moving parts
- Faster boot time

## Main Specifications

- Capacity: Measured in GB/TB
- Read Speed
- Write Speed
- Interface (SATA / NVMe)

## Learning Notes

SSDs improve system performance because they provide faster access to data compared to traditional HDDs.
