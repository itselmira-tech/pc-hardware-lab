# BIOS / UEFI

## What is BIOS?

BIOS is firmware that starts the computer hardware during the boot process.

## What is UEFI?

UEFI is the modern replacement for BIOS.
It provides more features and better hardware support.

## Main Functions

- Hardware initialization
- Checking connected components
- Starting the boot process
- Managing boot settings

## BIOS vs UEFI

BIOS:
- Older technology
- Limited features

UEFI:
- Modern firmware
- Supports larger drives
- Faster boot process

## Learning Notes

BIOS/UEFI is the first software that runs when a computer starts.
