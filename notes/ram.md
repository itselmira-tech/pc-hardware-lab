# RAM (Random Access Memory)

## What is RAM?

RAM is a temporary memory used by the computer while programs are running.

## Characteristics

- Fast access memory
- Temporary storage
- Data is lost when the computer is turned off

## Main Specifications

- Capacity: Measured in GB
- Speed: Measured in MHz
- Type: DDR4 / DDR5

## RAM vs Storage

RAM:
- Temporary memory
- Used for active programs

Storage:
- Permanent memory
- Used for saving files

## Learning Notes

More RAM allows a computer to run more applications at the same time.
