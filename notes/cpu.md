# CPU (Central Processing Unit)

## What is CPU?

The CPU is the main processing unit of a computer.
It executes instructions and performs calculations.

## Main Specifications

- Core: Number of processing units inside CPU
- Thread: Virtual processing capability
- Clock Speed: Measured in GHz
- Cache: Fast memory inside CPU

## Common CPU Brands

- Intel
- AMD

## Learning Notes

A better CPU can improve performance in multitasking, gaming, and professional applications.
