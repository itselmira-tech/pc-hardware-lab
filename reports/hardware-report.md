# Hardware Inspection Report

## Project Information

**Project:** PC Hardware Lab

**Purpose:** Hardware inspection and system documentation

**Operating System:** Windows 11

---

# Executive Summary

This report documents the inspection and analysis of a personal computer using Windows built-in diagnostic tools. The objective is to identify the installed hardware components, understand their functions, and demonstrate the ability to document technical information professionally.

---

# System Overview

The inspected computer is equipped with an Intel Core i5 processor, 16 GB of RAM, a Samsung SSD, and HP BIOS firmware. The system is suitable for everyday productivity tasks, software development, and educational laboratory exercises.

---

# Hardware Components

## Processor (CPU)

The processor is responsible for executing instructions and managing system operations.

**Installed CPU**

- Intel Core i5-8265U
- 4 Physical Cores
- 8 Logical Processors

### Observation

The processor provides sufficient performance for office applications, virtualization, programming, and hardware laboratory exercises.

---

## Memory (RAM)

Installed Memory:

- 16 GB

### Observation

16 GB of RAM allows efficient multitasking and provides enough memory for development tools and virtual machines.

---

## Storage

Installed Storage:

- Samsung SSD

### Observation

The SSD significantly improves boot time, application loading, and overall system responsiveness compared to traditional HDDs.

---

## BIOS / UEFI

The system firmware initializes hardware during startup before the operating system loads.

### Observation

The BIOS information was verified using Windows System Information.

---

# Boot Process Overview

The startup sequence follows these stages:

1. Power On
2. BIOS / UEFI Initialization
3. POST (Power-On Self Test)
4. Boot Loader
5. Windows Boot Manager
6. Operating System Startup

---

# Inspection Tools

The following Windows tools were used:

- System Information (msinfo32)
- Task Manager
- Windows Settings
- File Explorer

---

# Findings

The inspection confirmed that:

- Hardware components are correctly detected.
- System memory is functioning normally.
- SSD storage is operating properly.
- BIOS information is accessible.
- No abnormal hardware issues were identified during inspection.
