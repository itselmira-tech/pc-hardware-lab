# Hardware Troubleshooting Scenarios

## Scenario 1

### Problem

Computer does not boot.

### Possible Causes

- Faulty RAM
- Loose power cable
- Power supply failure

### Diagnostic Steps

- Check power cable
- Reseat RAM modules
- Verify motherboard LEDs

### Result

The RAM module was not fully seated.

---

## Scenario 2

### Problem

System is very slow.

### Possible Causes

- HDD instead of SSD
- Background applications
- Low available RAM

### Solution

- Close unnecessary applications.
- Upgrade storage to SSD if needed.
- Monitor memory usage.

---

## Scenario 3

### Problem

High CPU temperature.

### Diagnostic Steps

- Check CPU fan.
- Remove dust.
- Replace thermal paste if necessary.

### Result

Cooling performance improved after cleaning.
