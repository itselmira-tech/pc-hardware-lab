```text
                     |
                     v

           +--------------------+
              |      |       |

             CPU    RAM     SSD
              |
           Cooling Fan
```

## Description

The motherboard connects all major hardware components.

- CPU performs calculations.
- RAM stores temporary data.
- SSD stores the operating system and files.
- Power Supply delivers electrical power.
